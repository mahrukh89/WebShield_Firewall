Custom Firewall with Web GUI
Overview
This project is a custom firewall built using Bash (iptables) and Python (Flask). It provides both command-line control and a simple web interface to manage firewall rules.

Features
Packet filtering and stateful rules
NAT (Network Address Translation)
Logging and monitoring
Web-based firewall management

Description
The firewall is built on a Linux system and uses iptables to control traffic through packet filtering, stateful rules, and NAT. A Python-based web interface allows users to view firewall activity and manage rules without relying on the command line.

Purpose
The project demonstrates core firewall concepts and provides a simple, user-friendly way to control and monitor network security.
